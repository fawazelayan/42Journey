# Net Practice

## Description
NetPractice is a project focused on understanding the fundamentals of networking. The goal is to solve 10 levels of configuring IP addresses, subnet masks, and routing tables, with each level being harder than the one before it.
The project simulates how data packets travel through routers, switches, and the internet, requiring a deep understanding of how devices communicate within and across different networks.

## Instructions
How to run:
1. Download the net_practice.1.9.tgz file provided by 42.
2. Extract the file then enter the exctracted folder.
3. Run this command "sh run.sh"
4. Navigate to http://localhost:49242 in your web browser.

How to Export:

* Once a level is marked as OK, click the "Get my config" button.
* This will download a json file.

Submission Requirements:
* The repository must contain exactly 10 config files at the root (level1-level10).

## Resources

Networking Concepts Studied
* TCP/IP Addressing: Assigning a unique ID to hosts.
* Subnet Masks: Defining the boundaries of a network and calculating usable IP ranges.
* Default Gateways: Configuring the "exit point" for traffic leaving a local network.
* Routing Tables: Defining the next hop for specific destination networks to ensure two-way communication.
* OSI Model: Specifically focusing on Layer 3 (Network Layer).
